# Documentação do Frontend (React)

## Tecnologias
- **Build Tool**: Vite.
- **Biblioteca**: React 18.
- **Visualização 3D**: Three.js / React Three Fiber.

## Portais
1. **Admin**: Visão macro do sistema e gestão de planos.
2. **Laboratório**: Dashboard operacional para entrada de amostras.
3. **Produtor**: Portal do cliente final com relatórios e o **Digital Twin**.

## Componentes Chave
- `DigitalTwin.jsx`: Renderização 3D do crescimento das plantas baseado nos dados de solo.
- `api.js`: Camada de serviço centralizada com interceptação de token JWT.
